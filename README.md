# node_web_crawler
Write a scraper in node.js that will fetch all internal urls from all pages of medium.com (recursively) and save them into a file

## Tech Stack
 
 * Node
 * cherio


## Prerequisites

``` 
    npm: '7.0.14',
    node: '15.3.0',
   ```
  
 ## Install
 ``` 
    npm install request
    npm install cherio
  ```
  
 ## RUN
  ```
    node app.js
